# 常用类

Java 类库提供了不少常用类，可以在编程中直接调用使用。本节讲重点讲解 Arrays、StringBuilder、Calendar、Date、Math、System、Random 类以及类的常用方法

知识点

-   Arrays
-   StringBuilder
-   Calendar
-   Date
-   Math
-   System
-   Random

# Arrays

> 示例文件：[ArraysDemo](./day25/ArraysDemo.java)

Arrays 类包含用于操作数组的各种方法（例如排序和搜索）。还包含一个静态工厂，允许将数组转为 List
